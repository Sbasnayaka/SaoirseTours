\# Saoirse Tours CMS



A simple Tourism Content Management System built with PHP and MySQL.



\## Tech Stack

\- PHP 7.4+

\- MySQL

\- Apache (XAMPP)



\## Setup Instructions

1\. Copy project to `htdocs`

2\. Create database `tourism\_cms`

3\. Import `database.sql`

4\. Update `includes/config.php`

5\. Run: http://localhost/SaoirseTours/



\## Admin Login

\- Username: admin

\- Password: password



\## Note

This repository contains frontend + backend logic.

Database credentials are excluded for security.
